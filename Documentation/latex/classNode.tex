This function adds the input new\+Element\+Id (int) to the list of elements connected by this node, updating the mass, and weights of mass per connected element in the process.

First the mass before addition of the new element is recorded. Then the \hyperlink{classNode_a63e510fc9158eb15e751861bc14eae38}{Node\+::mass} is updated.

Each of the already recorded weights (in \hyperlink{classNode_a7ffccd2a70d08c1a42d4701f32a266c5}{Node\+::connected\+Element\+Weights}) of the connected elements (in \hyperlink{classNode_a18bae606efb025cc90b4c117776e0bf9}{Node\+::connected\+Element\+Ids}) will be updated with the scale new\+Mass / old\+Mass.
